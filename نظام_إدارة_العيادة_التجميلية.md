# نظام إدارة العيادة التجميلية - عرض تقديمي

## 📋 نظرة عامة على النظام

نظام متكامل لإدارة عيادة تجميلية يدمج إدارة العملاء، الحجوزات، الموظفين، والفواتير مع دعم كامل للطباعة الحرارية.

- **Flask** - إطار العمل الأساسي
- **SQLite** - قاعدة البيانات
- **Bootstrap 5** - واجهة المستخدم

# 🎉 خاتمة

نظام متكامل لإدارة العيادات التجميلية يجمع بين القوة التقنية وسهولة الاستخدام، مصمم خصيصًا لاحتياجات السوق العربي مع دعم كامل للغة العربية والطباعة الحرارية.




### نظام نقاط الولاء للعملاء
- تُعطى العملاء نقاطًا لكل جلسة أو دفعة
- يمكن استبدال النقاط بخصومات أو جلسات مجانية
- يعزز الولاء وتكرار الزيارات
### 2. تذكيرات المواعيد الآلية
- إرسال تذكيرات عبر واتساب أو رسائل SMS قبل الجلسات (مثل 24 ساعة قبل)
- تقليل حالات الإلغاء أو التأخر
- بالفعل يوجد كود واتساب، يمكن تطويره ليتم تلقائيًا
### 3. إدارة المخزون للمنتجات
- تتبع المنتجات المستخدمة في الجلسات (مثل الكريمات، الزيوت، إلخ)
- تنبيه عند انخفاض المخزون عن حد معين
- إضافة طلبات شراء جديدة للمخزون
### 4. تقييمات ومراجعات الخدمات
- السماح للعملاء بتقييم الخدمات والجلسات
- عرض المراجعات لتحسين الجودة
- إظهار التقييمات لموظفي المركز
### 5. تقارير مالية مفصلة
- تقارير شهرية/سنوية مع رسوم بيانية (مخططات)
- مقارنة الفترات المختلفة
- تفاصيل الدخل والخروج بشكل مرئي
### 6. لوحة متابعة أداء الموظفين
- تتبع عدد الجلسات لكل موظف
- إجمالي الإيرادات الناتجة عن كل موظف
- تقييم رضا العملاء عن كل موظف
### 7. تخصيص الباكدج (الباقات)
- إنشاء باكدج مخصصة لكل عميل
- إضافة خيارات إضافية للباكدج (مثل جلسات إضافية أو منتجات مجانية)
- مرونة في تسعير الباكدج
### 8. تاريخ طبي للعملاء
- تخزين معلومات طبية مهمة (حساسية، أمراض، تفضيلات)
- ضمان سلامة العملاء أثناء الجلسات
- تنبيه الموظفين لأي معلومات مهمة قبل البدء
### 9. بوابة حجز عبر الإنترنت
- صفحة عامة للعملاء لحجز الجلسات من المنزل
- اختيار الموظف والوقت المناسب
- تأكيد الحجز تلقائيًا
### 10. دعم الفروع المتعددة
- إدارة أكثر من فرع في نظام واحد
- تقارير منفصلة لكل فرع أو موحدة
- نقل العملاء بين الفروع بسهول
